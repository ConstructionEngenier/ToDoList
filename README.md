# ToDoList
___
## Description
**ToDoLst** is the productivity tool you need to get work and life organized. Collect tasks, organize projects, and 
plan 
your day.
___
## Stack
- Python 3.9
- Django 4.1.2
- PostgresQL
___
## How to run
- Install poetry (_pip install poetry_)
- Install dependencies (_poetry install_)
